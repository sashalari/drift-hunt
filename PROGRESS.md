## 🔑 КЛЮЧЕВЫЕ МОМЕНТЫ

### Архитектура игры
- **Canvas 2D** (HTML5) — текущий, работает
- **PlayerCar** — машина игрока с дрифт-механикой
- **BotCar** — простой AI с случайными поворотами
- **GameManager** — главный менеджер состояния (menu, playing, results, shop, leaderboard)
- **UIManager** — отрисовка экранов (меню, HUD, результаты, лидерборд, магазин)
- **SoundManager** — Web Audio API для звуков

### Система сохранения (localStorage)
```javascript
playerName          → Имя игрока
playerRating        → Рейтинг (1000+)
leaderboard         → Топ-100 игроков
ownedSkins          → Куплённые скины ['red', 'blue', 'yellow']
selectedSkin        → Активный скин ('red', 'blue', 'yellow')
analytics           → Метрики (D1, D3, D7, количество гонок)
```

### Разработка новых фич
1. Добавить метод в GameManager или UIManager
2. Добавить состояние ('shop' → 'dailyQuests' и т.д.)
3. Обновить update() для обработки состояния
4. Обновить render() для отображения
5. Протестировать в браузере
