🏎️ ГОНОЧНАЯ ИГРА — ПРОЕКТ ОЛЕГА И ЛЕВЫ

Обзор проекта

Цель: Разработать сетевую гоночную игру, где Лева и Олег зарабатывают деньги на продажах.

Целевая аудитория: Школьники района, друзья, одноклассники (онлайн).

📋 СТРУКТУРА ПРОЕКТА

ЭТАП 1: ПЛАН А (MVP — 1-2 недели)
  ├─ Быстро запустить на Scratch
  ├─ 2 трассы, макс 2 бота
  ├─ Заработок: 5-20k₽/месяц
  └─ ЦЕЛЬ: Протестировать, собрать фидбек

ЭТАП 2: ПЛАН B (FULL — 3-4 недели)
  ├─ Перезагрузка на Roblox (сетевая)
  ├─ 10 трасс, 4 бота, профиль, дрифт
  ├─ Заработок: 50-200k₽/месяц
  └─ ЦЕЛЬ: Масштабирование и большой доход

  • СПЕЦИФИКАЦИЯ_ИГРЫ.md — Полное описание всех фич
  • ПЛАНАMVP.md — Детальный план первого этапа (Scratch)

  • МЕХАНИКА_ИГРЫ.md — Детали управления и физики

🎮 Быстрый старт

Для Олега: 1. Открыть ТАСКЛИСТЫОЛЕГ.md 2. Следовать пошагово (неделя за неделей) 3. Писать код на Scratch (ПЛАН А) или Lua (ПЛАН B)

Для Левы: 1. Открыть ТАСКЛИСТЫЛЕВА.md 2. Рисовать и дизайнить согласно спецификации 3. Подготавливать графику для кодера

Для вас: 1. Отслеживать прогресс (1 раз в неделю) 2. Помогать, когда застрянут 3. Проверять финмодель и доходы
