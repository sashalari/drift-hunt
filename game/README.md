# 🏎️ PAKO 3D — Arcade Racing Game

Интерактивная 3D гоночная игра в стиле **Pako Highway** от первого лица с механикой **дрифта и погони**.

## 🎮 Характеристики

✅ **3D графика** (Three.js, WebGL)  
✅ **Реалистичная физика** — управление через угол поворота  
✅ **Дрифт-механика** — дополнительная скорость при боковом скольжении  
✅ **AI соперники** — полиция и обычные машины с умным AI  
✅ **Low Poly стиль** — яркие неоновые цвета и геометрические формы  
✅ **Гараж** — выбор машины (BMW, Audi, Mercedes, Jeep) и цвета  
✅ **Оффлайн режим** — работает без интернета (PWA)  
✅ **Мобильная версия** — полная поддержка сенсорных экранов  

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| **← →** | Повороты влево/вправо |
| **Shift** | Дрифт (боковое скольжение) |
| **E** | Нитро-ускорение |

## 🚀 Как запустить

### Локально (на компьютере)
```bash
# Способ 1: Просто открыть в браузере
open INDEX.html
# или в Windows: INDEX.html (двойной клик)

# Способ 2: Локальный сервер (для тестирования PWA)
python3 -m http.server 8000
# Открыть http://localhost:8000
```

### На мобильном
1. Скопировать файл INDEX.html на телефон
2. Открыть в браузере (Chrome, Safari, Firefox)
3. Или добавить на главный экран как приложение

### На GitHub Pages (для друзей)
```bash
# 1. Создать новый репозиторий на github.com
# 2. Загрузить файлы
# 3. Включить Pages в Settings → Pages
# 4. Открыть: https://username.github.io/repo-name/INDEX.html
```
