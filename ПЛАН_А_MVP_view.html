📋 ПЛАН А: MVP БЕЗОПАСНЫЙ (1-2 недели)

🎯 Цель этапа

Быстро запустить рабочую версию на Scratch, которую можно: - ✅ Протестировать с друзьями - ✅ Начать зарабатывать (5-20k₽) - ✅ Собрать фидбек для ПЛАНА B

⚙️ Технические детали

Параметр Значение
Платформа Scratch (Олег уже знает)
Экспорт HTML5 → Itch.io (монетизация)
Трассы 2 штуки (Пустыня, Город)
Боты Макс 2 машины
Управление Стрелки (влево/вправо, газ)
Дрифт НЕТ (добавим в ПЛАНЕ B)
Повороты НЕТ (прямая дорога или 90°)
Профиль Минимальный (имя + деньги)
Реклама Admob баннер между гонками
IAP Скины машин (20-100₽)
Сроки 1-2 недели интенсива

📆 ГРАФИК ПО НЕДЕЛЯМ

НЕДЕЛЯ 1: ОСНОВНАЯ МЕХАНИКА

ПН-ВЕ (Олег кодит)

День 1-2: Основы - [ ] Создать сцену (дорога, машинка, финиш) - [ ] Управление машинкой (стрелки влево/вправо) - [ ] Газ/тормоз (вперёд/назад) - [ ] Границы дороги (стены которые останавливают)

День 3-4: Боты и гонка - [ ] Создать 1 бота (красная машина) - [ ] Бот едет с постоянной скоростью - [ ] Определить финишную линию - [ ] Определить кто первый финишировал

День 5: Интеграция - [ ] Проверить, что всё работает без багов - [ ] Протестировать управление - [ ] Убедиться, что финиш считается правильно

ЛД (Лева рисует)

Параллельно: - [ ] Рисует две машины (синяя = игрок, красная = бот) - [ ] Рисует дорогу (простая, прямая) - [ ] Рисует финишную линию - [ ] Рисует меню (главное, паузу)

НЕДЕЛЯ 2: МОНЕТИЗАЦИЯ И ТЕСТИРОВАНИЕ

ПН-ВЕ (Олег кодит)

День 1-2: Награды - [ ] Система монет (1 место = 1000, 2 место = 500) - [ ] Показывать награду после гонки - [ ] Сохранять общее количество монет

День 3: Вторая трасса - [ ] Скопировать первую трассу - [ ] Переделать дизайн (город вместо пустыни) - [ ] Добавить выбор трассы на главном меню

День 4: Прогрессия - [ ] Уровень 1: гонка без ботов - [ ] Уровень 2+: с ботами - [ ] Сохранять текущий уровень

День 5: Экспорт - [ ] Экспортировать из Scratch в HTML5 - [ ] Залить на Itch.io - [ ] Настроить Admob (реклама)

ЛД (Лева рисует)

Параллельно: - [ ] Интерфейс (кнопки, текст) - [ ] Второй вариант машины (желтая) - [ ] Иконки (медали за награды) - [ ] Оформление финишной линии

💰 ФИНМОДЕЛЬ ПЛАН А

Доход от рекламы (Admob)

100 активных игроков/день
  × 5 игр в день (среднее)
  × $0.5 за 1000 просмотров
  = 250 рублей/день за рекламу
  = 7,500₽/месяц

Доход от скинов (IAP)

100 игроков
  × 20% хотят купить скин (20 игроков)
  × 50₽ средняя цена
  = 1,000₽ с гонки
  × 30 дней = 30,000₽/месяц

Реалистично: 5-20k₽/месяц (в зависимости от трафика)
