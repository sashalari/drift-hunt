# 💰 ФИНМОДЕЛЬ ИГРЫ

## ПЛАН А: MVP (Scratch) — 1-2 недели

### Источники дохода

#### 1️⃣ Реклама (Admob)

```
Модель: CPM (за 1000 просмотров)
Ставка: $0.5 за 1000 просмотров (в России)
Конверсия: 50% игроков смотрят рекламу
```

**Расчёт:**

```
Сценарий 1: Консервативный
├─ 100 активных игроков/день
├─ 5 гонок в день (среднее)
├─ 500 просмотров рекламы/день
├─ 500 просмотров × $0.5/1000 = $0.25/день
└─ $0.25 × 28 = $7/месяц = 420₽/месяц

Сценарий 2: Средний
├─ 500 активных игроков/день
├─ 5 гонок в день
├─ 2,500 просмотров рекламы/день
├─ 2,500 × $0.5/1000 = $1.25/день
└─ $1.25 × 28 = $35/месяц = 2,100₽/месяц

Сценарий 3: Оптимистичный
├─ 1,000 активных игроков/день
├─ 5 гонок в день
├─ 5,000 просмотров рекламы/день
├─ 5,000 × $0.5/1000 = $2.50/день
└─ $2.50 × 28 = $70/месяц = 4,200₽/месяц
```

#### 2️⃣ IAP Скины машин

```
Модель: Платные косметические предметы
Цена: 20-100₽ за скин
Конверсия: 5-20% купят минимум один скин
Средняя чек: 50₽ (если купил)
```

**Расчёт:**

```
Сценарий 1: Консервативный
├─ 100 активных игроков
├─ 5% хотят купить скин = 5 игроков
├─ 50₽ × 5 = 250₽/день
└─ 250₽ × 28 = 7,000₽/месяц

Сценарий 2: Средний
├─ 500 активных игроков
├─ 10% хотят купить скин = 50 игроков
├─ 50₽ × 50 = 2,500₽/день
└─ 2,500₽ × 28 = 70,000₽/месяц

Сценарий 3: Оптимистичный
├─ 1,000 активных игроков
├─ 15% хотят купить скин = 150 игроков
├─ 50₽ × 150 = 7,500₽/день
└─ 7,500₽ × 28 = 210,000₽/месяц
```

#### 3️⃣ IAP Премиум (отключить рекламу)

```
Модель: Одноразовая покупка
Цена: 50-200₽
Конверсия: 2-5% от новых игроков
```

**Расчёт:**

```
Сценарий 1: Консервативный
├─ 50 новых игроков/день
├─ 2% платят премиум = 1 игрок/день
├─ 100₽ × 1 = 100₽/день
└─ 100₽ × 28 = 2,800₽/месяц

Сценарий 2: Средний
├─ 200 новых игроков/день
├─ 3% платят премиум = 6 игроков/день
├─ 100₽ × 6 = 600₽/день
└─ 600₽ × 28 = 16,800₽/месяц

Сценарий 3: Оптимистичный
├─ 400 новых игроков/день
├─ 5% платят премиум = 20 игроков/день
├─ 100₽ × 20 = 2,000₽/день
└─ 2,000₽ × 28 = 56,000₽/месяц
```

## 💡 ОПТИМИЗАЦИЯ ДОХОДОВ

### Что работает хорошо?

✅ **Скины машин** — самый большой источник доходов  
✅ **Премиум подписка** — стабильный резидуальный доход  
✅ **Реклама** — всегда есть базовый доход  
✅ **Платные трассы** — дополнительный доход (но не главный)

### Как увеличить доход?

1. **Качество игры** → больше игроков → больше платящих
2. **Рекламная кампания** (в школе, соцсетях) → быстрее распространение
3. **Сезонные события** → лимитированные скины (люди спешат купить)
4. **Лидерборд** → люди платят за преимущество (компетитивность)
5. **Реферальная система** → каждый приведённый друг = % от его покупок

## 🎯 КРИТИЧЕСКИЕ ПОКАЗАТЕЛИ

| Показатель | Цель | Как измерить |
|---|---|---|
| **DAU** (игроков в день) | 500+ | Статистика Itch.io / Roblox |
| **Retention** (возврат) | 50%+ | % игроков вернувшихся на 2-й день |
| **Conversion** (платящие) | 5-10% | % от всех игроков что платили |
| **ARPU** (средний доход на игрока) | 50-200₽ | Общий доход / количество DAU |
| **LTV** (жизненная стоимость игрока) | 500-2000₽ | Сколько один игрок заработает за всё время |

## ⚠️ РИЗИКИ И КРАШ-ТЕСТЫ

### Что если игра не популярна?

```
Сценарий: Только 50 активных игроков
  ПЛАН А: 1-5k₽/месяц
  ПЛАН B: 5-20k₽/месяц
  
Решение:
  ├─ Улучшить качество (баги, оптимизация)
  ├─ Агрессивная реклама (в школе, соцсетях)
  ├─ Добавить новые фичи (события, турниры)
  └─ Если не помогает → переходим на другой продукт
```

### Что если Robux платежи не работают?

```
Решение: Параллельно использовать другие системы
  ├─ Admob для рекламы
  ├─ Паттреон для платящих поклонников
  ├─ Steam если перейдём на ПК
  └─ App Store если будет мобильная версия
```
